# vue-tutorial

## 概要

vue の練習環境

### 入っている機能

- DogAPI
- GoogleMap

### 使用技術

- vue-router
- vuetify
- vuex
- axios
- Google Maps JavaScript API Loader

## 構成

root
├ src
│ ├ api:
│ ├ assets
│ ├ components:
│ ├ constants:
│ ├ plugins:
│ ├ router: ルーティング情報
│ ├ views: ページ
│ └ lib/
├ .devcontainer: 開発環境
└ public

## 起動方法

### GoogleMap を利用する場合

`.env.sample`を参考に`.env`を作成する

### Docker 環境

`reopen in container`

### 共通

`yarn`

`yarn serve`
